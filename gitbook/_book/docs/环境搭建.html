环境搭建

环境搭建

安装 Node.js 因为 GitBook 是使用 Node.js 开发的，需要通过 Node.js 包管理工具 NPM 安装，所以在开始之前要先把 Nodejs 安装好。

Node.js 下载地址：https://nodejs.org/zh-cn/

NPM（node package manager），通常称为 node 包管理器。使用 NPM 可以对 node 包进行安装、卸载、更新、查看、搜索、发布等操作。

安装完 Node.js，NPM 就可以直接用了。

使用 NPM 安装 GitBook 的命令行工具： npm install gitbook-cli -g

安装完成后查看 GitBook 的版本： gitbook --version

更新 GitBook命令： $ npm update gitbook-cli -g

卸载 GitBook 命令： $ npm uninstall gitbook-cli -g
